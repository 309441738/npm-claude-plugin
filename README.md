# 邮轮产品 NPM 工具 · Claude Code 插件

通过 NPM 使用邮轮产品 API，开发者可以在自己的应用中集成邮轮产品信息。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/npm-claude-plugin
/plugin install npm@npm-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **邮轮产品 NPM 工具**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
